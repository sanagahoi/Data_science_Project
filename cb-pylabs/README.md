# Py-Lab: Machine Learning Journey 🚀

A collection of Jupyter Notebooks covering fundamental to advanced Machine Learning concepts using Python, Scikit-Learn, and Pandas.

## 📊 Project Overview

| Category | Notebook | Description |
| :--- | :--- | :--- |
| **Classification** | [Spam Classifier](CodeBasic_MNB_SpamClassifier.ipynb) | Email spam detection using Multinomial Naive Bayes. |
| **Classification** | [Titanic Survival](CodeBasic_NB_titanic.ipynb) | Predicting passenger survival using Naive Bayes. |
| **Classification** | [Iris Flower SVM](CodeBasic_SVM_Iris.ipynb) | Classifying flower species using Support Vector Machines. |
| **Classification** | [Digit Recognition](CodeBasic-Multiclassification_Load_digits.ipynb) | Multi-class classification of handwritten digits. |
| **Regression** | [Home Price Prediction](CodeBasic_Prepro_handlingData_HomePrice.ipynb) | Handling missing data and predicting real estate prices. |
| **Clustering** | [Income K-Means](CodeBasic_KMeans_income.ipynb) | Grouping data points by income levels using K-Means. |
| **Validation** | [K-Fold Cross Validation](CodeBasic_KFoldCrossValidation_load_digits.ipynb) | Evaluating model performance with K-Fold techniques. |

## 🛠️ Technologies Used
* **Python**
* **Jupyter Notebooks**
* **Scikit-Learn** (SVM, Naive Bayes, Random Forest, K-Means)
* **Pandas & NumPy** (Data Manipulation)
* **Matplotlib & Seaborn** (Visualization)
